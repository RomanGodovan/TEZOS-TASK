Задание. Cоздать контракт с такими требованиями:

Должен содержать метод SendRequest, который принимает в качестве параметра (address).
Метод SendRequest должен вызывать метод get_info() контракта, чей адрес передается как параметр в метод SendRequest. 
Метод get_info принимает в качестве параметра contract(nat*nat)(контракт ожидающий пару).
Созданный вами контракт должен принимать ответ от вызванного контракта и сохранять в storage результат. 
Если второй элемент пары четное число, то его следует заменить на число 33 перед сохранением.

Метод  SendRequest должен иметь такие ограничения:

Может быть вызван не больше одного раза.
Может быть вызван только владельцем контракта.

Контракт должен быть создан в testnet сети Delphi и нужно провести тестовая транзакция с вызовом метода SendRequest с параметром. Адрес контракта который необходимо вызвать: KT1WYEpgbT3sobMQmYCaWf8kvffrzLJCukFw.
